# thrift-ai
Thrift Ai is a Java-based application designed to help users manage their finances by tracking income and expenses, storing this data in a database, and providing AI-driven suggestions on where to spend and where to cut back.

#Features
  -User-Friendly Interface: Easy-to-use GUI for entering income and expenses.
  -Secure Data Storage: Stores financial data securely in a database.
  -AI-Powered Insights: Analyzes spending patterns using AI and offers personalized suggestions.
